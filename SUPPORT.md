# Support

This repository is a portfolio/demo project and does not provide guaranteed support.

## Getting help

If you have a question or suggestion:

1. Search existing Issues first.
2. Open a new Issue using the templates:
   - Bug report
   - Feature request

## Security issues

Please do not open public Issues for security vulnerabilities.  
Follow the process in [SECURITY.md](SECURITY.md).

## What to include in Issues

- What you were trying to do
- Expected vs actual behaviour
- Steps to reproduce
- Logs/screenshots (redact secrets)
- Your environment (OS, Python, Docker, Terraform, cloud target)
